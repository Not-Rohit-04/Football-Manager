# Football Manager ⚽

Football Manager is a Flask-based web application that allows users to browse football players, manage player data, and build their own dream team from a database of legendary and modern footballers.

## Features

### Current Features (V1)

- View all football players
- Player detail pages
- Admin login system
- Protected admin routes
- Add new players
- Edit player information
- Delete players
- Build a custom team
- Team rating calculation
- Responsive Bootstrap interface

### Coming Soon (V2)

- Hall of Fame Teams
- Mix Team Generator
- User Registration & Login
- Save Custom Teams
- Team History
- Team Comparison System

### Future Plans

The long-term goal of Football Manager is to become an interactive football simulation game where users can:

- Build custom football teams
- Save and manage multiple squads
- Compete against AI-controlled teams
- Simulate football matches
- Compare team ratings and chemistry
- Track wins, losses, and trophies
- Create leagues and tournaments
- Build ultimate dream teams from different eras

## Tech Stack

- Python
- Flask
- SQLAlchemy
- SQLite
- Bootstrap 5
- HTML
- CSS

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Create and activate a virtual environment

```bash
python -m venv .venv
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser and visit

```text
http://127.0.0.1:5000
```
