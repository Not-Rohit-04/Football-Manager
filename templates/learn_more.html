Info

Football Manager is a web application built using Python, Flask, SQLAlchemy, SQLite, HTML, CSS, and Bootstrap.

The project allows users to browse a database of football players and build their own team by selecting players from different positions. The application provides an easy way to view player information and create a balanced squad.

Current Features

View Players Browse all available players stored in the database, including their position and other details. Build Your Team Create your own football team by selecting players from the available squad.

Admin

Admin Management Administrators can securely log in to manage the player database.

Admin features include:

Add new players
Edit existing player information
Delete players from the database
Update player details

Technologies Used

Python
Flask
SQLAlchemy
SQLite
HTML5
CSS3
Bootstrap 5

Project Goal

The purpose of this project is to practice full-stack web development concepts including database management, CRUD operations, authentication, and dynamic web page rendering using Flask.
Future versions may include transfers, team statistics, match simulations, and league management systems.
