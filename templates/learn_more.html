Info

Football Manager is a web application built using Python, Flask, SQLAlchemy, SQLite, HTML, CSS, and Bootstrap.

About The Project

Football Manager is a Flask-based football simulation game built using Python, Flask, SQLAlchemy, Bootstrap, HTML, and CSS.

The game allows users to create a 5-a-side football team using randomly generated players from different positions. Every player has their own rating, club, nationality, and age, making each team unique.

After building a squad, players can compete against a computer-generated team in simulated football matches where team ratings help determine the outcome.

⚽ Generated Players

Explore a database filled with randomly generated football players, each with unique ratings, clubs, positions, and nationalities.

🧠 Team Builder

Select players from different positions and build the strongest possible 5-a-side squad.

🏆 Match Simulation

Challenge computer-generated teams and see whether your squad has what it takes to win.

Future Plans

Future versions may include saved teams, match history, league systems, tournaments, advanced statistics, and additional gameplay features.
