About The Project

Football Manager is a Flask-based football simulation game built using Python, Flask, SQLAlchemy, Bootstrap, HTML, and CSS.

The game allows users to create a 5-a-side football team using randomly generated players from different positions. Every player has their own rating, club, nationality, and age, making each team unique.

After building a squad, players can compete against a computer-generated team in simulated football matches where team ratings help determine the outcome.
